== ServerSocket 생성

* accept는 client가 연결이 설정될 때까지 대기 합니다.

[source,java]
----
ServerSocket serverSocket = new ServerSocket(8080);

for(;;){
    //client 접속할 때 까지 대기
    Socket client = serverSocket.accept();
}
----
* 서버가 수락(accept)하면 읽기(BufferedReader), 쓰기(BufferedWriter)를 할 수 있습니다.
